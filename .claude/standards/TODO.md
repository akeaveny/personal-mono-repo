# TODO Vault Standards

## BOARD.md Section Order

Sections must appear in this order:

1. `## Daily Focus`
2. `## In Progress`
3. `## Up Next`
4. Category sections (e.g., `## Trips`, `## Personal Finance`, `## Weekly Reminders`)
5. `## Inbox`
6. `## Backlog`
7. `## Completed / Removed`

New category sections go between "Up Next" and "Inbox".

## Tagging

### Inline hashtags

Tags go at the end of the task title, before the `—` source attribution:

```
- [ ] Task title #tag1 #tag2 — *via Source*
```

### Available tags

| Tag | Purpose |
|---|---|
| `#trip` | Travel — flights, hotels, trip planning |
| `#personal-finance` | Banking, taxes, insurance, cards |
| `#weekly` | Recurring weekly items |

Custom hashtags can be added freely.

### Auto-tagging rules (applied by `/refresh-board`)

Only **new** tasks added to Inbox are auto-tagged. Existing tasks are never re-tagged.

| Keywords (case-insensitive) | Tag |
|---|---|
| flight, flights, hotel, airbnb, trip, travel, ireland, vacation, passport | `#trip` |
| tax, taxes, RBC, CRA, bank, card, claim, finance, penalty, scotia | `#personal-finance` |
| weekly, reminder, recurring, every week | `#weekly` |

A task can receive multiple tags if it matches keywords from multiple rules.

## Task identifiers

Preserve these identifiers on task lines to keep `/refresh-board` merge matching working:

- **Google Tasks**: The exact task title

## Files

- `TODO/TODO.md` — Auto-generated by `/sync-todos`. Never edit manually.
- `TODO/BOARD.md` — User-curated. `/refresh-board` only adds to Inbox and flags removed tasks.

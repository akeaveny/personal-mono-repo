# 2025 Tax Filing Checklist

**Tax year:** January 1 – December 31, 2025
**Filing deadline:** April 30, 2026
**RRSP contribution deadline:** March 2, 2026

---

## Employment Income

- [ ] T4 — Helios → `slips/employer/`
- [ ] Pay stubs / earnings statement → `slips/employer/`

## Investment Income (Wealthsimple)

- [ ] T3 — Trust income (Non-registered CAD `HQ23FVP06CAD`) → `slips/wealthsimple/`
- [ ] T5 — Investment income (Non-registered USD `HQ56WWN07USD`) → `slips/wealthsimple/`
- [x] T5008 — Securities transactions (Non-registered CAD `HQ23FVP06CAD`) → `slips/wealthsimple/`
- [ ] T5008 — Securities transactions (Non-registered USD `HQ56WWN07USD`) → `slips/wealthsimple/`
- [ ] RRSP contribution receipt (`HQ23FVR46CAD`) → `slips/wealthsimple/`
- [x] T4 FHSA — First Home Savings Account (`HQ53CYG67CAD`) → `slips/wealthsimple/`
- [ ] Foreign asset verification report → `slips/wealthsimple/`

## Banking (RBC)

- [ ] T5/NR4 — Interest & investment income → `slips/rbc/`

## Deductions & Credits

- [ ] Donation receipts → `slips/donations/`
- [ ] RRSP deduction limit (from 2024 NOA or CRA My Account)
- [ ] T2151 — Direct transfer request (if applicable) → `slips/other/`

## Filing

- [ ] File return (Wealthsimple Tax / StudioTax / accountant)
- [ ] Save filed return PDF → `returns/`
- [ ] Save Notice of Assessment (NOA) → `returns/`

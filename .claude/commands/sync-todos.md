Sync all open tasks from Google Tasks and Google Calendar into `TODO/TODO.md`. This is a full replace — regenerate the entire file each run.

## Steps

### 1. Fetch Google Tasks

Call `mcp__google-tasks__list` to get all tasks from Google Tasks.

Extract from each task: `title`, `due` (if present), `notes` (if present), `status`, and the task list name. Only include tasks that are not completed (status is not `completed`).

If this step fails, continue and include an HTML comment: `<!-- Google Tasks sync error: {error message} -->`

### 2. Fetch Google Calendar Events

Call `mcp__google-calendar__list-events` with:
- `calendarId`: `primary`
- `timeMin`: today's date at midnight (ISO 8601)
- `timeMax`: 7 days from today at end of day (ISO 8601)

This fetches upcoming events for the next 7 days. Extract from each event: `summary`, `start` (date or dateTime), `end` (date or dateTime), `location` (if present), `htmlLink`.

If this step fails, continue and include an HTML comment: `<!-- Google Calendar sync error: {error message} -->`

### 3. Generate TODO/TODO.md

Create the `TODO/` directory if it does not exist. Write `TODO/TODO.md` with the following format:

```markdown
# TODOs

> Last synced: YYYY-MM-DD HH:MM ET

## Google Tasks

- [ ] Task title (due: YYYY-MM-DD)
- [ ] Task title with no due date

(repeat for each incomplete task; skip completed tasks)

## Google Calendar — Next 7 Days

### YYYY-MM-DD (Day of Week)

- **HH:MM – HH:MM** | Event summary | 📍 Location
- **All day** | All-day event summary

(group events by date; show time in local format; include location only if present)
```

At the bottom of the file add:

```markdown
---
*Auto-generated by `/sync-todos`. Do not edit manually — changes will be overwritten on next sync.*
```

### Important Notes

- Always fully replace `TODO/TODO.md` — never append.
- If all sources fail, still write TODO.md with the error comments and empty sections.
- Use the current time in ET for the "Last synced" timestamp.
- Sort Google Calendar events chronologically by start time, grouped by date.
- For Google Calendar all-day events, use the `start.date` field; for timed events, use `start.dateTime`.
